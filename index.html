3D Data Formats

Common Formats


Every 3D file format stores information about the geometry of a 3 dimensional object. Each extension has its own characteristics, and some can store additional information, such as appearance information (colour, texture), animation, and the scene (camera position, lighting).


These are some of the most commonly used 3D file formats:

  • .STL (StereoLitography)
  • .OBJ (Object Wavefront)
  • .STEP
  • .FBX
  • .3DS
  • .USD

Desired Outcome


Since each format has a different way of encoding and can store different aspects of a 3D shape, it is important to keep in mind the end result when working on a 3D model, to ensure that the extension we use will fit our needs accordingly.


There are mainly 4 uses for 3D modeling:

  • 3D Printing
  • Engineering
  • Gaming
  • Animation
